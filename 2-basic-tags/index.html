Here us a paragraph with some text made important for the reader (defaults to bold) as well as some text emphasised for the reader (defaults to italic). Notice how the indenting of the code is change here to allow for easier legibility by a human. (machines don't care so much. HTML is very 'loose' and does not care about spacing or case. Even when you make mistakes the browser will still try and render the page)

the 'hr' tag defines a thematic change in the content, and defaults to drawing a horizontal rule across the page. The 'hr' tag does not need a closing tag.

The 'a' tag lets you link to other pages. See it used here to link to a HTML reference page. click here to view W3School.com's HTML reference page.
